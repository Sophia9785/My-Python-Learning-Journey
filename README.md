# 长方体体积计算器

## 项目简介
这是一个基于 Python 的**交互式命令行长方体体积计算器**。  
用户通过键盘输入长、宽、高（以空格分隔），程序即可实时输出对应的体积，并附带单位（立方厘米）。  
当用户输入 0 0 0 时，程序将优雅退出。

## 功能特性
- **循环交互**：持续等待用户输入，无需反复启动脚本。  
- **输入校验**：仅接受**正整数**；自动捕获并提示**格式错误**或**非正整数**情况。  
- **友好提示**：中文界面，清晰易懂的错误与操作提示。  
- **一键退出**：输入 0 0 0 即可安全退出程序。

## 运行环境
- Python ≥ 3.6（无需第三方库）

## 快速开始
1. 克隆或下载本项目到本地  
2. 打开终端（Windows 可用 CMD / PowerShell / VSCode 终端），进入项目目录  
3. 运行脚本  
   ```bash
   长方体体积计算器.py
   ```

4. 按提示输入长、宽、高，例如：  
   
```
   请输入长 宽 高（用空格分隔）：3 4 5
   ```

   程序将输出：  
   
```
   体积 = 3 × 4 × 5 = 60 立方厘米
   ```

5. 输入  0 0 0 退出程序

## 代码说明
- `print("长方体体积计算器")`：欢迎语  
- `while True:`：无限循环，持续等待用户输入  
- `shu = input(...).strip()`：读取用户输入并去除首尾空格  
- `if shu == "0 0 0": break`：退出指令  
- `a, b, c = map(int, shu.split())`：将输入拆分为三个整数  
- `if a <= 0 or ...:`：校验是否为正整数  
- `tiji = a * b * c`：计算体积  
- `except ValueError:`：排除非整数或格式错误
